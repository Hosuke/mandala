# 金胎不二 · 兩部曼荼羅引擎

> 絹本曼荼羅是投影，不是結構。結構是：一張圖、兩種遍歷方向、一套變換語義。
> 這個引擎承載結構本身。

瀏覽器打開即是壇城，無須構建、無須安裝、無外部圖片資源——一切尊形皆程序所現。

```
python3 -m http.server 8000   # 任一靜態服務即可
open http://localhost:8000
```

## 結構即教義：五層架構

| 層 | 檔案 | 教義對應 |
|---|---|---|
| **不二層** | `js/data/deities.js` | 單一規範圖數據，永不直接渲染。每尊一條記錄；胎藏（`t`）與金剛界（`k`）只是同一節點的兩個面。一條記錄同時持有兩面者，即兩部同體之尊——金胎不二的錨點（大日、五佛對應、普賢即金剛薩埵、觀音即金剛法、文殊即金剛利、虛空藏即金剛寶、彌勒即金剛因）。 |
| **投影層** | `js/layout.js` | 同一圖之兩個布局函數：胎藏＝同心放射（流出之拓撲，共時的）；金剛＝九宮序列（轉換之拓撲，歷時的）。九會中八會是「回響」——同一批節點經表示變換（種字／三昧耶形／微細／供養／忿怒）之再現，變換函子的可見化。 |
| **相之層** | `js/textures.js` | 一切尊形 canvas 程序生成：月輪、種字（法曼荼羅）、三昧耶形線描圖庫四十種（三昧耶曼荼羅）、忿怒焰環。 |
| **遍歷層** | `js/traversal.js` | 上轉門／下轉門：同一回路之二向。金剛界行九會之螺旋（成身→……→降三世三昧耶，及其逆），胎藏行諸環之漲落（中臺→外金剛部，及其逆）。 |
| **行者層** | `js/camera.js` | 俯瞰（觀畫）與入壇（在場）。入壇之際諸尊立起——法曼荼羅轉為羯磨曼荼羅，畫成為空間。 |

## 操作

- **不二滑尺**（底部）：胎藏 ⇄ 金剛之連續形變。中點即「不二」——兩部同體之錨點微明可辨；缺一面之尊攝入其部主而隱顯。
- **下轉**：佛成為世界之路。自中心順旋而出，法語隨行。
- **上轉**：眾生成佛之路。自外逆旋而入，同一條路，倒讀。
- **入壇**：第一人稱入於壇中（拖曳環顧，WASD 行步，Esc 出）。
- **點尊**：得種字、梵名、真言、部族、兩部所在之位。

## 尊位凡例

尊位依現圖系而有所省略：成身會具足三十七尊，胎藏取中臺八葉並十一院代表尊，共八十三節點。種字從通行本，諸傳承間或有異說。本引擎所現介於法曼荼羅與三昧耶曼荼羅之間，入壇則向羯磨曼荼羅——四種曼荼羅，亦是表示之異，非所表之異。
